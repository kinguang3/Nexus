使用项目虚拟环境运行单个Python文件：

```bash

d:\Nexus\venv\Scripts\python.exe d:\Nexus\Data\文件名.py

# 示例：运行 vector_stores.py
d:\Nexus\venv\Scripts\python.exe d:\Nexus\Data\vector_stores.py

# 示例：运行 knowledge_base.py
d:\Nexus\venv\Scripts\python.exe d:\Nexus\Data\knowledge_base.py

# 启动Streamlit应用（自动切换虚拟环境）
python d:\Nexus\Data\run_app.py
```
